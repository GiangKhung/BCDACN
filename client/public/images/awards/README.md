# Hướng dẫn thêm Logo Giải thưởng

## Thư mục này chứa logo các giải thưởng

Vui lòng thêm 4 file logo với tên chính xác như sau:

1. **propertyguru-asia-awards.png** - Logo PropertyGuru Asia Property Awards
2. **propertyguru-business.png** - Logo PropertyGuru For Business
3. **property-report.png** - Logo Property Report
4. **asia-real-estate-summit.png** - Logo Asia Real Estate Summit

## Yêu cầu hình ảnh:
- Kích thước khuyến nghị: 200x60px hoặc tỷ lệ tương tự
- Định dạng: PNG (nền trong suốt) hoặc SVG
- Dung lượng: < 100KB mỗi file
- Logo nên rõ ràng, có thể là grayscale hoặc màu

## Đường dẫn file:
```
client/public/images/awards/
├── propertyguru-asia-awards.png
├── propertyguru-business.png
├── property-report.png
└── asia-real-estate-summit.png
```

## Hiệu ứng:
- Logo mặc định: Grayscale 50%
- Khi hover: Hiển thị màu gốc (grayscale 0%)
- Opacity giảm nhẹ khi hover

## Đường dẫn đầy đủ:
```
E:\BCDACN\client\public\images\awards\
```

Sau khi thêm logo, refresh trình duyệt để thấy thay đổi!
