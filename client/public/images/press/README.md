# Hướng dẫn thêm Logo Báo chí

## Thư mục này chứa logo các báo chí

Vui lòng thêm 6 file logo báo với tên chính xác như sau:

1. **tuoi-tre.png** - Logo Tuổi Trẻ
2. **vnexpress.png** - Logo VnExpress
3. **dan-tri.png** - Logo Dân Trí
4. **cafef.png** - Logo CafeF
5. **vietnamnet.png** - Logo VietnamNet
6. **thanh-nien.png** - Logo Thanh Niên

## Yêu cầu hình ảnh:
- Kích thước khuyến nghị: 120x40px hoặc tỷ lệ tương tự
- Định dạng: PNG (nền trong suốt) hoặc SVG
- Dung lượng: < 50KB mỗi file
- Logo nên rõ ràng, dễ nhận diện

## Đường dẫn file:
```
client/public/images/press/
├── tuoi-tre.png
├── vnexpress.png
├── dan-tri.png
├── cafef.png
├── vietnamnet.png
└── thanh-nien.png
```

## Gợi ý tải logo:
Bạn có thể:
1. Tìm logo chính thức từ website của các báo
2. Sử dụng công cụ như Google Images (tìm "logo [tên báo] png")
3. Hoặc tải từ các nguồn logo miễn phí

## Đường dẫn đầy đủ:
```
E:\BCDACN\client\public\images\press\
```

Sau khi thêm logo, refresh trình duyệt để thấy thay đổi!
