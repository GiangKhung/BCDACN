# Thư mục Avatar Cá nhân Môi giới

## Hướng dẫn sử dụng

1. **Thêm avatar:**
   - Copy ảnh chân dung vào thư mục này
   - Đặt tên file theo format: `agent-1.jpg`, `agent-2.png`, etc.
   - Kích thước khuyến nghị: 150x150px hoặc 300x300px (vuông)

2. **Định dạng hỗ trợ:**
   - JPG/JPEG (khuyên dùng)
   - PNG
   - WebP

3. **Ví dụ:**
   ```
   agents/
   ├── pham-cong-tin.jpg
   ├── le-dinh-minh-hai.jpg
   ├── nguyen-truong.jpg
   ├── nguyen-tran-thai-son.jpg
   ├── tran-van-minh.jpg
   └── pham-thi-huong.jpg
   ```

4. **Sử dụng trong code:**
   ```javascript
   avatar: '/images/agents/pham-cong-tin.jpg'
   ```

## Lưu ý
- Tối ưu kích thước file (< 200KB)
- Ảnh chân dung rõ nét, chuyên nghiệp
- Nền sáng, trang phục lịch sự
- Tỷ lệ 1:1 (vuông) để hiển thị đẹp
