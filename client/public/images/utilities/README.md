# Hướng dẫn thêm hình ảnh cho Tiện ích

## Thư mục này chứa các icon cho phần "Hỗ trợ tiện ích"

Vui lòng thêm 4 hình ảnh với tên file như sau:

1. **xem-tuoi-xay-nha.png** - Icon cho "Xem tuổi xây nhà"
2. **chi-phi-lam-nha.png** - Icon cho "Chi phí làm nhà"
3. **tinh-lai-suat.png** - Icon cho "Tính lãi suất"
4. **tu-van-phong-thuy.png** - Icon cho "Tư vấn phong thủy"

## Yêu cầu hình ảnh:
- Kích thước khuyến nghị: 64x64px hoặc 128x128px
- Định dạng: PNG (có nền trong suốt) hoặc JPG
- Dung lượng: < 50KB mỗi file

## Đường dẫn file:
```
client/public/images/utilities/
├── xem-tuoi-xay-nha.png
├── chi-phi-lam-nha.png
├── tinh-lai-suat.png
└── tu-van-phong-thuy.png
```

## Sau khi thêm hình:
Code đã được cấu hình sẵn để tự động load hình từ thư mục này.
Không cần thay đổi code, chỉ cần đặt đúng tên file!
