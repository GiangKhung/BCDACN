# Hướng dẫn thêm Logo Doanh nghiệp

## Thư mục này chứa logo các doanh nghiệp tiêu biểu

Vui lòng thêm 8 file logo với tên chính xác như sau:

1. **thien-minh-capital.png** - Logo Thiên Minh Capital
2. **hausland.png** - Logo Hausland
3. **cityland.png** - Logo CityLand
4. **hoang-the-group.png** - Logo Hoàng Thế Group
5. **kim-tinh-group.png** - Logo Kim Tính Group
6. **cat-tuong-group.png** - Logo Cát Tường Group
7. **novaland.png** - Logo Novaland
8. **vingroup.png** - Logo Vingroup

## Yêu cầu hình ảnh:
- Kích thước khuyến nghị: 200x80px hoặc tỷ lệ tương tự
- Định dạng: PNG (nền trong suốt) hoặc JPG (nền trắng)
- Dung lượng: < 100KB mỗi file
- Logo nên có màu sắc rõ ràng (sẽ có hiệu ứng grayscale khi hover)

## Đường dẫn file:
```
client/public/images/partners/
├── thien-minh-capital.png
├── hausland.png
├── cityland.png
├── hoang-the-group.png
├── kim-tinh-group.png
├── cat-tuong-group.png
├── novaland.png
└── vingroup.png
```

## Hiệu ứng:
- Logo mặc định: Grayscale (đen trắng)
- Khi hover: Hiển thị màu gốc
- Có shadow và transform nhẹ khi hover

