# Thư mục Logo Công ty

## Hướng dẫn sử dụng

1. **Thêm logo công ty:**
   - Copy file logo vào thư mục này
   - Đặt tên file theo format: `company-1.png`, `company-2.jpg`, etc.
   - Kích thước khuyến nghị: 120x120px hoặc 240x240px (vuông)

2. **Định dạng hỗ trợ:**
   - PNG (khuyên dùng - có nền trong suốt)
   - JPG/JPEG
   - SVG
   - WebP

3. **Ví dụ:**
   ```
   companies/
   ├── victory-real.png
   ├── phu-thanh.jpg
   ├── khang-dien.png
   └── hoang-quan.png
   ```

4. **Sử dụng trong code:**
   ```javascript
   logo: '/images/companies/victory-real.png'
   ```

## Lưu ý
- Tối ưu hóa kích thước file (< 100KB)
- Sử dụng nền trong suốt cho PNG
- Đảm bảo logo rõ nét, chất lượng cao
